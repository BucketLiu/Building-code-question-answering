# Building Code Data

This folder contains the building code documents used in this study, including parsed Markdown files and the LLM fine-tuning dataset.

## Contents

- Parsed code documents in Markdown format (after processing by CodeReader)
- `Alpaca_dataset.json` — QA pairs for LoRA fine-tuning (6,210 pairs)

## Data Sources

Six hydraulic building codes are included, covering concrete dams, earth-rockfill dams, and construction specifications. These files are used to build the vector knowledge base for retrieval and to fine-tune the language model.
